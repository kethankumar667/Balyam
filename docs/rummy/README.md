# Bhalyam Rummy — Nostalgia UX Docs

Design + roadmap for re-shaping Bhalyam Rummy into a heart-touching, nostalgia-first multiplayer experience for Indian players and their diaspora.

Rummy only. No code changes here — these are planning docs. Implementation lands in [`client/src/games/rummy/`](../../client/src/games/rummy/) and [`server/src/games/rummy/`](../../server/src/games/rummy/).

## Explicit non-goals

- No competitor cloning.
- No real-money mechanics, FOMO loops, predatory progression, or casino tropes.
- No arcade flash. The anchor is the **family card table**, not the lounge.
